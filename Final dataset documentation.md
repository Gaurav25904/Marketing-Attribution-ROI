Final Dataset Notes

 Overview

The final dataset is created after cleaning and processing raw marketing data.  
It is used for dashboard development and KPI analysis.

---

Data Processing Steps

1. Collected raw marketing data  
2. Handled missing values  
3. Removed duplicate records  
4. Standardized timestamp format  
5. Applied attribution models:
   - First Touch Attribution
   - Last Touch Attribution
   - Linear Attribution  

---
Final Dataset Structure

The dataset is aggregated at user and campaign level.

Key Metrics Included:
- Total Conversions  
- Total Revenue  
- Total Ad Spend  
- Total Clicks  
- Total Impressions  

---

Calculated Metrics

- ROAS (Return on Ad Spend)
  = Revenue / Ad Spend  

- CAC (Customer Acquisition Cost)
  = Ad Spend / Conversions  

---

Purpose of Dataset

This dataset is used to:
- Build Power BI / Tableau dashboard  
- Analyze campaign performance  
- Compare attribution models  
- Support business decision-making

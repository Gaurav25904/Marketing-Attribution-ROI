Data Dictionary

 Dataset Name: Cleaned Marketing Dataset

This dataset contains user-level marketing data used for attribution analysis and dashboard creation.



 Columns Description

| Column Name  | Description |
| User_ID | Unique ID assigned to each user |
| Timestamp | Date and time of user interaction |
| UTM_Source | Source of traffic (Google, Facebook, etc.) |
| Campaign | Name of the marketing campaign |
| Converted | Conversion status (1 = Converted, 0 = Not Converted) |
| Revenue | Revenue generated from the user |
| Ad_Spend | Cost spent on advertisement |
| Clicks | Number of clicks by the user |
| Impressions | Number of times the ad was shown |


Data Notes

- Dataset is cleaned and preprocessed
- Missing values handled
- Duplicate records removed
- Data combined from multiple marketing sources
- Used for attribution modeling (First Touch, Last Touch, Linear)

 Purpose

This dataset is used to:
- Analyze marketing performance
- Calculate KPIs like ROAS and CAC
- Build dashboard for decision making
